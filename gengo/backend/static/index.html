Gengo API Server

This is the API server for the Gengo language learning application.

API Documentation

GET /api/health

Check if the API server is up and running.

POST /api/auth/login

Log in to the application.

GET /api/users/me

Get the current user's profile.

GET /api/sessions/recent

Get recent practice sessions.

POST /api/speech/analyze

Upload and analyze speech recording.

Frontend Development

The frontend application is served separately using Trunk.

Run cd ../frontend && trunk serve to start the frontend development server.
